# Bus Seat Reservation System

A simple console-based project built in C++ that demonstrates 
the use of classes and objects (OOP). 

## What it does
- Add new bus routes with driver name, timings, and locations
- Reserve a seat (1-32) for a passenger on any bus
- View all 32 seats of a bus and see which are taken
- Display all available buses in a table format

## How to run
Compile with: g++ seat_reservation.cpp -o seat_reservation
Run with: ./seat_reservation

## Built with
- C++
- Standard Library only (iostream, string, iomanip)
